Ultimate Tic Tac Toe — Play or Download

Play a polished browser version of the game instantly, or download and run the Python 3D version locally. The Python version is more advanced (3D, OpenGL, richer AI) but must be run on your machine — it won't run inside GitHub Pages.

Modes:

Classic, Connect (6×7), Ultimate (9×9 fallback), 3D (4×4×4)

Browser play uses the 2D advanced build. The Python game includes 3D mode and OpenGL (if available).

Note: If you want the Python 3D experience, follow the README instructions (download, setup venv, pip install -r requirements.txt, then python main.py).

Run the Python game (local)

  1. Create venv: python -m venv .venv
  2. Activate: source .venv/bin/activate (mac/linux) or .venv\\Scripts\\activate (Windows)
  3. Install deps: pip install -r requirements.txt
  4. Run: python main.py
If the 3D mode doesn't appear the game will automatically fall back to a high-quality 2D mode. NumPy recommended for audio.
